# Lorentz's Theory
A 3 dimensional parkour game made in Unreal Engine 5 about escaping a man-made structure built to contain you.
This is both my first 3D videogame and my final year project for secondary school. It includes parkour movement, a large sense of scale and a surreal environment, as well as great performance for an Unreal Engine game. You can run this on a school laptop.
Right now it's more of a short demo/prototype but i'm working on expanding the game and adding polish.
See controls in the main menu.
